# 💳 Credit Card Fraud Detection System

## 📌 Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques. The dataset is highly imbalanced, and special techniques like data preprocessing and resampling are used to improve model performance.

---

## 🚀 Features

* Data preprocessing and cleaning
* Handling imbalanced dataset (SMOTE / undersampling)
* Feature scaling
* Model training using ML algorithms
* Performance evaluation using metrics
* Visualization of results

---

## 🧠 Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn
* Imbalanced-learn (SMOTE)
